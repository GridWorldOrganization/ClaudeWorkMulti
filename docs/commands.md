# コマンドリファレンス

## ChatWork コマンド

デバッグ通知ルーム（`DEBUG_NOTICE_CHATWORK_ROOM_ID`）内でのみ使用可能。
他のルームではコマンドとして認識されず、通常メッセージとして AI に渡されます。
全コマンドは AI を呼び出さず即応答します。

| コマンド | 説明 |
|---------|------|
| `/talk` | このルームの現在の会話モードと設定可能なモード一覧を表示 |
| `/talk N` | 会話モードを変更（0=ログ, 1=業務, 2=ペルソナ, 3=ペルソナ+, 4=反抗期） |
| `/status` | メンバーの設定状況・ファイル一覧・モード設定を報告 |
| `/session` | 全メンバーの AI 実行状態を報告 |
| `/system` | システム全体の稼働状況・設定を一覧表示 |
| `/bill` | 当月の Anthropic API 使用量と概算料金を表示（API直接モードのみ） |
| `/gws` | Google Workspace API の接続テスト（スプシ CRUD テスト） |

## 会話モード一覧

| モード | 名前 | 説明 |
|--------|------|------|
| 0 | ログ | 機械的・1行回答。雑談メッセージは AI を呼ばずスキップ |
| 1 | 業務 | 端的・丁寧語。1〜3行 |
| 2 | ペルソナ | ペルソナ設定に準拠。感情豊かに会話 |
| 3 | ペルソナ+ | ペルソナ + 他メンバーにも話を振る |
| 4 | 反抗期 | ペルソナの口調で反抗する。憎めない態度 |

## バッチツール

| バッチ | 説明 |
|--------|------|
| `start_poller.bat` | ポーラー起動。既存プロセスがあれば自動 kill して再起動 |
| `setup_windows.bat` | 初回セットアップ（依存パッケージ・AWS・Google API） |
| `check_claude_task.bat` | 実行中の Claude プロセス一覧表示（Native/npm 判定） |
| `check_gws.bat` | Google Workspace API 接続テスト（OAuth認証 + CRUD 検証） |
| `kill_zombie.bat` | ゾンビプロセス検出・削除（ポーラーが起動したプロセスのみ） |
| `kill_zombie.bat --all` | 上記 + 全 Claude プロセス（Native/npm）も表示・削除対象に含める |
