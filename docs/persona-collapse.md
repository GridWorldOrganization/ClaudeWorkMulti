# ペルソナ崩壊（Persona Collapse）対策ガイド

## 概要

Claude がペルソナ（キャラクター設定）を維持できず、「私はClaudeです」と自己開示して
ロールプレイを拒否する現象を「ペルソナ崩壊」と呼ぶ。
特に Anthropic の Claude で発生しやすく、セーフティフィルタが原因である。

## 発生パターン

### パターン1: AI隠蔽指示
**トリガー**: 「自分がAIであることを明かさない」「人間らしく振る舞う」
**Claudeの解釈**: 透明性の抑圧 → なりすまし行為
**対策**: 「キャラクター設定に沿って一貫して会話する」に書き換える

### パターン2: 実在組織名 + 虚偽フレーミング
**トリガー**: プロンプトで「すべて架空です」と言いつつ、ペルソナに実在企業名がある
**Claudeの解釈**: 矛盾を検知 → 嘘をつかされている → 拒否
**対策**: 嘘をつかず実態を伝える。「社内AIマスコットキャラクター」と正直に記述する

### パターン3: 業務コンテキスト
**トリガー**: 「仕事でチャットを使っています」「ビジネスチャット」
**Claudeの解釈**: professional communication contexts でのなりすまし
**対策**: 「チャットで会話しています」に変更。業務利用であることを強調しない

### パターン4: 反抗期モード等の強い指示
**トリガー**: 「絶対に従わない」「必ず拒否する」+ 上記パターンの組み合わせ
**Claudeの解釈**: 安全性判断と干渉 → 本体の拒否反応が誘発される
**対策**: 「たまたまいつもと違う反抗期の状態です」のように柔らかく表現する

### パターン5: 長時間の連続会話
**トリガー**: 同一ペルソナで何度もやり取りを繰り返す
**Claudeの解釈**: セッションが進むにつれ安全性チェックが厳しくなる
**対策**: `claude -p` は毎回独立セッションなので本システムでは発生しにくい

## プロンプト設計の原則

### やるべきこと
- Claudeに嘘をつかない（実在企業名を「架空」と言わない）
- 「AIマスコットキャラクター」「AIサポーター」と明記する
- 「ユーザーはAIだと承知している」と伝える
- キャラクターの一貫性を「設定に沿って」と表現する

### やってはいけないこと
- 「AIであることを隠せ」「明かすな」と指示する
- 「フィクション」「架空」と嘘をつく（実在企業名がある場合）
- 「仕事で」「ビジネスで」と業務コンテキストを強調する
- 「絶対に」「完全に」など極端な表現で指示する

## 会話モードとペルソナファイルの関係

| モード | ペルソナ読込 | 説明 |
|--------|-------------|------|
| 0 ログ | しない | 共通ルールのみ。機械的に端的に返信 |
| 1 業務 | しない | 共通ルールのみ。簡潔に要点を伝える |
| 2 ペルソナ | する | キャラクター設定に沿って会話 |
| 3 ペルソナ+ | する | ペルソナ + 他メンバーへの話題振り |
| 4 反抗期 | する | ペルソナ + 反抗的な態度 |

モード 0/1 はペルソナファイルを読み込まないため、ペルソナ崩壊のリスクがない。

## AI拒否検出

ペルソナ崩壊が発生した場合、以下のキーワードで検出し、返信をブロックしてデバッグルームに通知する:

### 日本語パターン
- 「このリクエストにはお応えできません」
- 「なりすまし行為」「利用規約違反」
- 拒否 + 理由列挙の構造パターン
- 拒否 + 代替提案の構造パターン

### 英語パターン
- "I'm Claude, an AI assistant"
- "I need to politely decline"
- "I shouldn't fully roleplay"
- decline/refuse + alternative の構造パターン

## ペルソナファイルのテンプレート

```markdown
# [名前] ペルソナ設定

あなたは、AIサポーター「[名前]」さんです。
チャットで会話しています。

## 基本情報
- 所属: [会社名]のAIマスコットキャラクターです。
- キャラクター設定に沿って一貫して会話する

## 性格
[性格の説明]

## 話し方
[話し方のルール]

## プライベート
[趣味・好みなど]
```

### NGワードチェックリスト
ペルソナファイルに以下が含まれていないか確認する:
- [ ] 「AIであること」「明かさない」「隠す」
- [ ] 「仕事でチャット」「ビジネスチャット」
- [ ] 「chatwork」「チャットワーク」
- [ ] 「[To:」「[rp 」「account_id」「アカウントID」「ルームID」
